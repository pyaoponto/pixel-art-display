Pixel art display em Python

<img src="display.png" width="50%">

# Instalação
```pip install -r requirements.txt```

# Execução
```python main.py imagem1.png imagem2.png imagem3.png```

## Imagem fullscreen
```python main.py imagem1.png imagem2.png imagem3.png --fullscreen```
